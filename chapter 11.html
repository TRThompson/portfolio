Chapter 11 - Data types

  • Document of all chapter 11 theory
    • KEY POINTS:
    • 5 data types: character, string, Boolean, integer and real.
    • All data types are represented in the computer in binary: 0s and 1s
    • Binary is a number system based on 2; the column values are the powers of 2 starting at 2⁰, 2², 2³ etc
    • Denary is our common number system; column values are powers of 10
    • Integers are represented in binary using powers of 2. Add up the column value where there is a 1 to get denary equivalent
    • To convert from denary divide by 2 repeatedly and write down the remainder at each stage or simply use column headings
    • Two ways to represent -ve numbers; ‘Sign and magnitude’ and ‘Two’s complement’
    • Sign and Magnitude: MSB is replaced by sign bit, 1 = -ve 0 = +ve.o
    • Split binary into nibbles then into Hex number
    • Divide by 16 to get denary into Hex

    Representing Text
    • Agreed method of representing characters
    • ASCII originally based on 8-bit binary pattern using 7 bits plus a single parity bit and could represent 128 seperate characters
    • Extended ASCII set uses 8 bits and can represent 256 characters
    • Unicode was originally a 16-bit restricted code allowing more than 65 000 characters but the restriction has been removed. The original ASCII is still included with the same values.

    Boolean data
    • True or false
    • 1 = True
    • 0 = False
    • Only requires one bit of storage but the values are often stored in one byte for convenience
